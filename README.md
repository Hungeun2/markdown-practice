# 표(Table)
position 속성

값 | 의미 | 기본값  
--|:--:|--:
static | 기준 없음 | O
relative | 요소 자신 | X
absolute | 위치 상 부모 요소 | X
fixed | 뷰포트 | X

기본은 왼쪽 정렬  
: 입력에 따라 좌 우 정렬 가능
<br/>복잡한 표는 만들 수 없다.
